# منصة آن للدعاية والإعلان – نموذج منصة متكاملة (واجهة + خادم)

هذا المشروع يمثل نموذجًا متكاملًا لمنصة **آن للدعاية والإعلان**:

- صفحة هبوط + واجهة كاملة في ملف واحد `index.html` تعمل مباشرة على GitHub Pages.
- تفعيل شكلي للحسابات: عميل، مصمم، مطبعة، إدارة (Admin).
- لوحات بسيطة لكل حساب تُدار عبر JavaScript في المتصفح (بدون قاعدة بيانات).
- مجلد `server/` يحتوي على خادم Node.js + Express كنموذج يمكن تشغيله محليًا لاحقًا.

## كيفية الاستخدام مع GitHub Pages

1. افتح الملف المضغوط (ZIP) بعد التحميل.
2. ارفع **كل الملفات والمجلدات داخل المشروع مباشرة** إلى مستودع GitHub (بدون وضعها داخل مجلد إضافي).
3. من إعدادات المستودع:
   - Settings → Pages
   - Source: الفرع `main`
   - Folder: `/ (root)`
4. ستحصل على موقع يعمل على:
   `https://USERNAME.github.io/REPO_NAME/`

> لا حاجة لأي أوامر `npm` لتشغيل الواجهة على GitHub Pages، لأن كل شيء مبني في `index.html`.

## تشغيل الخادم (اختياري – محلي)

إذا أردت ربط واجهة المنصة بخادم تجريبي:

```bash
cd server
npm install
npm start
```

سيتوفر خادم بسيط على:
`http://localhost:4000`

ويمكن تعديل الواجهة لاحقًا للاتصال به عبر AJAX أو fetch.
