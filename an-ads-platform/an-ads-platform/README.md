# منصة آن للدعاية والإعلان - نموذج أولي

هذا المستودع يحتوي على نموذج أولي (Prototype) لمنصة **آن للدعاية والإعلان** التي تربط بين:

- العملاء
- المصممين المعتمدين
- المطابع المعتمدة

## مكونات المشروع

- `server/` : خادم Node.js + Express (واجهة برمجية API مبدئية)
- `client/` : واجهة أمامية (Frontend) بسيطة باستخدام React + Vite

## طريقة التشغيل

### 1) تشغيل الخادم (Backend)

```bash
cd server
npm install
npm start
```

الخادم سيعمل افتراضياً على:  
`http://localhost:4000`

### 2) تشغيل الواجهة الأمامية (Frontend)

في نافذة أخرى:

```bash
cd client
npm install
npm run dev
```

ستجد الواجهة على:  
`http://localhost:5173` (أو ما يظهر لك في الطرفية)

## الأدوار (Roles)

في النموذج المبدئي هناك 3 أنواع من المستخدمين:

- `client` : عميل
- `designer` : مصمم
- `printer` : مطبعة

تسجيل الدخول الآن تجريبي (بدون قاعدة بيانات حقيقية)، ويتم عن طريق اسم مستخدم + دور فقط.

- هذا نموذج مبدئي يمكن تطويره لاحقاً:
